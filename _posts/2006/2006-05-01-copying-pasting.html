--- title: Copying & Pasting date: '2006-05-01T09:45:00.000+02:00' tags: modified_time: '2006-05-01T09:46:32.633+02:00' blogger_id: tag:blogger.com,1999:blog-15137028.post-114646959262231083 blogger_orig_url: http://www.continuousimprover.com/2006/05/copying-pasting.html --- While reading this article, I ran into two must-have tools which should have been part of Visual Studio from the beginning. First of all, Smart Paster is a handy tool for pasting a chunk of text into a source file in various nicely formatted formats. Secondly, Copy Source As HTML allows you to copy a code snippet for use in your blog or website while maintaining the syntax coloring.
