--- title: NDoc is over, welcome Sandcastle date: '2006-08-03T14:13:00.000+02:00' tags: modified_time: '2006-08-03T14:15:34.980+02:00' blogger_id: tag:blogger.com,1999:blog-15137028.post-115460733496500310 blogger_orig_url: http://www.continuousimprover.com/2006/08/ndoc-is-over-welcome-sandcastle.html --- Kevin Downs, the leader of the NDoc project has thrown the towel in the ring (is that English btw?). He claims to not have received any support from the community or from Microsoft. That’s a shame. I’ve been using NDoc since 2001, and I was eagerly waiting for the moment a .NET 2.0 version would become available.
Unfortunately, Microsoft’s Sandcastle project is far from complete. With the CTP, it is very possible to compile a set of assemblies in a nice .CHM file, but it will still take a few months more before we get the nice user interface NDoc offered. Moreover, to get it to work for .NET 1.1, you need to perform a decent amount of hacking.
